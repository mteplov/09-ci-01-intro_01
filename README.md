# Домашнее задание к занятию 7 «Жизненный цикл ПО»- Теплов Михаил

# Описание задания

Познакомьтесь с различными видами досок, которые предлагает Kaiten, пройдите весь путь задачи до её завершения.
---
# Что нужно сделать?

## 1. Настройка доски
- Создайте пространство и добавьте новую доску типа Scrum.
- Добавьте новую колонку в «Готово для тестирования».
- Переместите колонку так, чтобы она была после колонки «В работе».

---

# 2. Создание задачи по Frontend

## Создание задачи
Создайте задачу в Backlog доске:
```text
Frontend Bug
````
Переместите все задачи на доску Sprint, в колонку:
```text
Бэклог спринта
```
---
## Работа над задачей
Переместите одну из задач в колонку:
```text
В работе
```
Добавьте комментарий:
```text
Баг будет устранён путём обновления библиотеки в микрофронтенде
```
---
## Создание дочерней карточки
Создайте дочернюю карточку в Backlog доске:
```text
Обновить библиотеку
```
Название задачи:
```text
[LIB-456] Upgrade Material-UI from v4.12.3 to v5.0.0 in product-details microfrontend to resolve layout bugs
```
---
## Работа с дочерней задачей
Переместите новую карточку в:
```text
Бэклог Спринта
```
Затем в:
```text
В работе
```
Добавьте комментарий:
```text
Library Material-UI updated to v5.0.0
```
Добавьте трудозатраты:
```text
1 ч
```
После выполнения переместите задачу в колонку:
```text
Готово
```
---
## Завершение родительской задачи
Переместите задачу:
```text
Frontend Bug
```
в колонку:
```text
Готово
```
Добавьте трудозатраты:
```text
0.2 ч
```
---
# 3. Создание задачи по Backend
## Создание задач
Создайте две задачи в Backlog доске:
```text
Backend — Integration with YandexPay
```
---
## Создание дочерней задачи
Добавьте дочернюю задачу:
```text
Update API Endpoint for Improved Error Handling
```
Переместите подзадачу в колонку:
```text
Готово для тестирования
```
---
## Работа над backend-задачей
Переместите задачу в:
```text
В работе
```
Добавьте комментарий:
```text
Update the API endpoint to enhance error handling mechanisms.
```
Добавьте трудозатраты:
```text
4 ч
```
После этого переместите задачу в:
```text
Готово для тестирования
```
---

## Тестирование
В колонке:
```text
Готово для тестирования
```
добавьте комментарий:
```text
All tests passed successfully
```
Добавьте трудозатраты:
```text
1 ч
```
После успешного тестирования переместите задачу в:
```text
Готово
```
---
## Завершение backend-задачи
Переместите задачу:
```text
Поправить bug на Backend
```
в колонку:
```text
Готово
```
